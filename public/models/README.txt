Tutankhamun mask — same asset as on Sketchfab
==============================================

Source (CC Attribution 4.0 — credit amirfiala1):
https://sketchfab.com/3d-models/tutankhamun-gold-mask-992c57047cdf4412b970ad798d5ad0c9

Local Three.js file (optional — speeds up the hero and works offline):

  public/models/tutankhamun-mask.glb

How to get the GLB:

1. Open the Sketchfab link above (log in if download asks you to).
2. Click **Download 3D Model**.
3. Choose **glTF Binary (.glb)**.
4. Save/rename the file as **tutankhamun-mask.glb** in this folder.

If this file is missing, the site uses Sketchfab’s embedded viewer for the same model (online only).
